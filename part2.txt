1- What are Mutation Observers in JavaScript?
 MutationObserver is a built-in object that observes a DOM element and fires a callback in case of changes.
  meaning that it provides the ability to watch for changes being made to the DOM tree.

2- Explain 3 different CSS selectors.
    A.The "." class selector : selects all elements of a specific class to modify their style all at once.
    B.Attribute selector : selects all elements with a specified attribute within a class.
    C.Id selector : selects an element based on the value of its id.
